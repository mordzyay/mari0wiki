enemy properties

Main Page

REMINDER: This documentation is based on the Alesans Entities mod

Required:
Property Description/Function
"quadCount": Amount of quads (horizontal frames)
"width": Width of enemy hitbox (1 is 16 pixels/1 tile)
"height": Height of enemy hitbox (1 is 16 pixels/1 tile)
"quadCenterX": Center on the x axis of the quad (from left)
"quadCenterY": Center on the y axis of the quad (from top)
"offsetX": Sprite offset on x axis (from left)
"offsetX": Sprite offset on y axis (from top)

Common:
Property Description/Function
base= Base off of another custom enemy This must be outside of the json at the top!
"spawnOffsetX": Spawn offset on X axis (1 is 16 pixels/1 tile)
"spawnOffsetY": Spawn offset on Y axis (1 is 16 pixels/1 tile)
"movement": How will it move? (string)
"truffleshuffle" - Enemy moves left and right (think Goombas)
"truffleshuffleSpeed": movement speed (1 is 16 pixels/1 tile)
"truffleshuffleAcceleration": movement acceleration (1 is 16 pixels/1 tile)
"turnAroundOnCliff": Think red Koopas (true, false)
"piston" - Goes in and out (think Piranha Plant)
"pistonDistX": horizontal dist in X (1 is 16 pixels/1 tile)
"pistonDistY": vertical dist in Y (1 is 16 pixels/1 tile)
unfinished

Graphics:
Property Description/Function
"noSpriteSets": Disables spritesets (true, false)
"customScissor": Only draw part of an enemy (x, y, width, height)
"invertedScissor": Dont draw in that area (true, false)
"animationType": What animation method to use? (string)
"mirror" - If the enemy is mirrored (think Goombas)
"animationSpeed": time between mirroring
"quadNo": quad number to use
"frames" - If the enemy uses frames (think Koopas)
"animationStart": first quad
"animationFrames": number of frames
"animationSpeed": frame duration in secs.
"none" - If the enemy is static (think Podaboos)
"quadNo": quad number to use
